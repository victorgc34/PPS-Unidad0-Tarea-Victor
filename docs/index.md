# PPS – Unidad 0 – Tarea: Documentación del Proyecto

## **Introducción**

Este repositorio contiene la documentación y los archivos necesarios para la realización de la tarea de la Unidad 0 de la asignatura **Programación en el Puesto del Trabajo (PPS)**.

El objetivo principal es aplicar conocimientos relacionados con **Git, GitHub**, automatización mediante **GitHub Actions**, despliegue de documentación usando **MkDocs** y la creación de un servicio web con **Nginx** y **Docker**.

A lo largo del repositorio se documentan las distintas fases del proceso, desde la creación del repositorio hasta el despliegue final, siguiendo una estructura organizada y modular.

Esta documentación servirá como referencia del flujo completo de trabajo **DevSecOps** básico aplicado al proyecto.
