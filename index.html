PPS – Unidad 0 – Tarea: Documentación del Proyecto

Introducción

Este repositorio contiene la documentación y los archivos necesarios para la realización de la tarea de la Unidad 0 de la asignatura Programación en el Puesto del Trabajo (PPS).

El objetivo principal es aplicar conocimientos relacionados con Git, GitHub, automatización mediante GitHub Actions, despliegue de documentación usando MkDocs y la creación de un servicio web con Nginx y Docker.

A lo largo del repositorio se documentan las distintas fases del proceso, desde la creación del repositorio hasta el despliegue final, siguiendo una estructura organizada y modular.

Esta documentación servirá como referencia del flujo completo de trabajo DevSecOps básico aplicado al proyecto.

Índice de Documentación

A continuación se presenta un índice con enlaces al resto de documentos incluidos en este repositorio:

1. Git y creación del repositorio

  • git.md
    • Documentación del proceso de creación del repositorio y las principales operaciones realizadas con Git y GitHub.

2. Creación de WorkFlow de GitHub Actions y Vinculación con GitHub Pages

  • gitActions.md
    • Configuración de GitHub Actions que genera automáticamente la documentación con MkDocs. También se configura GitHub Pages para publicar esa documentación, quedando accesible la web a través del propio repositorio.

3. Creación de un contenedor Docker con NGINX

  • docker.md
    • Creación de un contenedor Docker Nginx a través de Docker Compose para poder visualizar la documentación generada por Github Actions.

4. Conclusión

  • conclusiones.md
    • En las conclusiones se reflexiona sobre el proceso completo de la tarea, destacando los aprendizajes obtenidos en Git, GitHub Actions, MkDocs y Docker, así como la importancia de integrar estas herramientas en un flujo de trabajo automatizado y reproducible.
